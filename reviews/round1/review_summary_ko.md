## Reviewer 1: 방법론 전문가

### 여전히 유효한 지적사항

1. **Jamo-LoRA 미구현**: 실제 실험은 모두 표준 LoRA(r=4, 16, 64)입니다. 논문 제목의 "Jamo-LoRA"(자모 토큰화 + 복합 손실 함수)는 아직 구현/실험되지 않았습니다.
   - **해결 방향**: Jamo-LoRA 실험 완료 후 추가하거나, 제목을 "한국어 ASR을 위한 LoRA 랭크 스케일링 분석"으로 변경

2. **의미적 일관성 메트릭 부재**: $L_{sem}$ (의미적 일관성 손실)이 실제로 구현되지 않음. 결과 JSON에 해당 메트릭 없음.
   - **해결 방향**: 해당 내용을 "향후 연구"로 이동하거나, 실제 구현 후 추가

3. **통계적 검증 부족**: 구성당 단일 실행, 신뢰 구간 없음, 유의성 검정 없음.
   - **해결 방향**: 다중 시드(3-5회) 실행으로 표준편차 보고

4. **하이퍼파라미터 명시 필요**: 학습률, 배치 크기, 에포크 수, LoRA alpha, 드롭아웃, 옵티마이저 등.
   - **해결 방향**: ✅ 갱신 버전에서 하이퍼파라미터 테이블 추가됨

5. **코드/모델 공개 없음**: 재현성을 위해 코드 저장소 필요.
   - **해결 방향**: GitHub 공개 계획 명시

## Reviewer 2: 실험/실증 전문가

**점수**: Reject → (갱신 후 재평가 필요)
**신뢰도**: 5/5

### 여전히 유효한 지적사항

1. **누락된 실험**:
   - 전체 미세조정(Full Fine-Tuning) 베이스라인 없음 → 상한선 비교 불가
   - 다른 PEFT 방법(Adapter, Prefix-Tuning) 비교 없음
   - 표준 한국어 ASR 벤치마크(KsponSpeech 등) 평가 없음

2. **랭크 스케일링 분석**: 실제 결과는 일관된 이야기를 보여줌:
   - r=4→r=16: **가장 큰** CER 감소 (0.94%p)
   - r=16→r=64: 수확체감 (0.41%p)
   - "이것은 실제로 흥미로운 결과이며, 논문이 정확히 보고하지 못한 것입니다"

3. **추론 지연시간 이상**: r=4(1.06s)가 베이스라인(1.35s)보다 빠르고, r=64(1.28s)가 r=16(1.86s)보다 빠른 것은 비단조적이며 설명이 필요.
   - **해결 방향**: 측정 분산, MPS 워밍업 효과 분석 필요

4. **효율성 데이터 부족**: 피크 메모리 사용량, FLOPs, 디스크 크기, 실시간 인자(RTF) 없음.

5. **SOTA 비교 필요**: KsponSpeech에서 Whisper Large-v3는 CER ~7-10%, 전문 한국어 ASR은 CER <5%. 맥락화 필요.

### 수정 권고

> "표준 LoRA 적응으로 Whisper Small CER을 13.18%에서 8.04%로 줄인 것은 유의미한 결과입니다."
> "정직한 보고와 함께 표준 벤치마크 비교가 추가되면 가치 있는 논문이 될 수 있습니다."
